# 🚀 WALKAROUND


Modern web-based platform developed to simplify and digitize the weekly inspection and ticket management process for managers and employees through a clean and efficient dashboard experience.

The system allows users to open tickets, submit required information easily, and track workflow processes digitally instead of relying on traditional paper-based or spreadsheet methods.

---

## ✨ Features

* Secure Authentication System
* Ticket Creation & Management
* Weekly Inspection Workflow
* Easy Information Submission
* Responsive Dashboard UI
* Cloud-Based Firebase Integration
* Multi-Page Navigation System

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript
* Firebase Authentication
* Firebase Hosting
* Firestore Database
* Git & GitHub
